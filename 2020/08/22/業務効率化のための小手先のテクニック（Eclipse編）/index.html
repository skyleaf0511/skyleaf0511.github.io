はじめに

メインコンテンツ

★:特におすすめ

■Eclipse
ショートカットキー
★Ctrl + Shift + R : プロジェクト内のソース検索
わざわざプロジェクト内の階層を辿らずに、直接ソースを開ける
ファイル名を大体把握したらかなり便利

  • Alt + ← / Alt + → : 直近の参照した行に戻る/進む
    スクロールしたり思い出したりする手間が省けて便利

  • Alt + Shift + ↑ / Alt + Shift + ↓ : メソッドの先頭に飛ぶ
    スクロールの手間が省けて便利

  • Ctrl + H : ワークスペース/プロジェクト内で文字列検索（拡張子も指定できる）
    断片的にしか分からない処理がどこにあるか、どのくらいあるか探すときなどに便利

  • Ctrl + F : ファイル内で文字列検索、置換
    置換は例えばメソッド内の変数名をすべて変えたいときに便利。正規表現を使った一括置換もできる
    選択した範囲の中から検索することも可能

  • Shift + Enter : 次の行の先頭にカーソルを移動
    例えば”abcdefg”の”d”の位置にカーソルがあっても、改行されずに次の行の先頭に飛べるので便利

  • Shift + D : カーソルがある行を削除
    わざわざ行全体を選択したり行の数字をクリックしなくても済むので便利

  • Ctrl + / : コメントアウト

タブの配置
ファイル名をドラッグするとタブの配置を変えられる
個人的にソースを上下に配置すると横まで見られるので便利
上下5段左右2列まで可能。でも上下3段、左右2列程度がおすすめ
（1つのファイルをよく見たくなったら、ファイル名をダブルクリックすると全画面に拡がる）
あとは必要ないタブは閉じたり最小化しておくほうが、ファイルの中身が見やすくておすすめ

アウトラインタブ
今見ているメソッドは何だっけ？というときに便利

その他

  • メソッドの中括弧をダブルクリックしてメソッド内を全選択
    その状態でCtrl + Fを押して検索すると探しているものが、メソッド内のどこにどのくらいあるか分かって便利
  • ワークスペースの階層を辿るときにファイルの先頭の文字列を打つと該当するファイルの位置に選択が移動する
    例
    ←　: 選択位置
    jsp>
    hoge1.jsp ←①この位置で1回クリックした状態
    hoge2.jsp
    hoge3.jsp
    ～～～～～～～～
    hoge99.jsp
    fuga1.jsp
    fuga2.jsp ←③どこかで再度1回クリックした後、「fuga2」と入力した状態
    fuga3.jsp
    ～～～～～～～～
    fuga99.jsp
    piyo1.jsp ←②「piyo」と入力した状態
    piyo2.jsp
    piyo3.jsp
    （Eclipse以外でもディレクトリ一覧があるものなら大体使える）

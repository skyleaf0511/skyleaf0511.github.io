メインコンテンツ

★:特におすすめ

■Windows
個人的よく使うショートカットキー

  • Ctrl + E : エクスプローラ起動
  • Ctrl + W または Alt + F4 : 閉じる
  • Ctrl + Shift + N : フォルダーを新規作成
  • F2 : ファイル名変更
  • Ctrl + Shift + ドラッグ : ショートカット作成
  • Win + (↑ / ↓ / ← / → ) : ウィンドウを移動する
  • prt sc (alt + prt sc) : ウィンドウのスクリーンショットを撮る
  • Win + shift + S : 指定した領域のスクリーンショットを撮る
  • ctrl + home / ctrl + end : ウィンドウの上下端に飛ぶ
    （Excelファイルを保存するときにctrl + homeを押すのは一部ではマナーになっている）
  • Shift + ctrl + ファイルをクリック : 別窓で開く

ブラウザのショートカットキー

  • Ctrl + T : 新しいタブを開く
  • Ctrl + 数字 : 左から数えてn番目のタブを開く
  • Ctrl + W : 閉じる

その他

  • よく使う記号や名前は単語登録すると便利
    例えば、現場ではセキュリティ上ブラウザ等でパスワードの自動入力をしてはいけないが、
    「ID　パスワード」を何らかの文字（列）に登録しておけば、手入力せずに切り取りと貼り付けで入力が可能。
    ※ID　パスワードは単語登録しないほうが良いものが多いので、これは勧めているわけではありません。全くのテスト用のIDパスワードならまず問題ないと思います）
    手順は
    ①メニューバーの辞書のアイコンを右クリック
    ②単語と読みを入力して品詞は「短縮よみ」を選択

★スタートアップアプリ
PC起動時に指定したファイルやソフトを自動で開く機能
メールクライアントやいつも使うディレクトリ、ファイルを指定しておくと便利
設定方法
①エクスプローラーのアドレスバーで「shell:startup」と入力
②開いたフォルダにファイルやソフトのショートカットを配置する
③Windowsの検索バーで「スタートアップ」と入力し、「スタートアップアプリ」を開く
④PC起動時に立ち上げたいファイルやソフトをオンにする

★PowerShell（ツール作成）
詳しくはPowerShellコマンド.txtを参照。でも下にも簡単な解説あり

PowerShellとは
Windows7以降のOSに標準搭載されたシェル、スクリプト言語。コマンドプロンプトの上位互換のようなもの
パイプライン（処理を変数に渡して扱う）が使えるため、コマンドプロンプトで書くバッチ（.bat）より記述の幅が拡がる

できること
Windowsの自動操作（ツール作成）
例えば複数のファイル/フォルダの作成、名前変更、移動、コピー、削除、実行などを自動化できる
（拡張すればMicrosoftクラウドとの連携もできる。クラウドは恐らくほとんどの現場では使えないが）

起動するには以下のいずれかの場所で”powershell”と入力する

  • ファイル名を指定して実行（Win + R）
  • エクスプローラーのアドレスバー　（ここで起動すると、今いるディレクトリでpowershellを起動可能）
  • コマンドプロンプト

実行ファイルを作るには、
テキストファイルにPowerShellを記述して、拡張子”.ps1”で保存する

“.ps1”を実行するには、以下のどちらかをする必要がある

  • “.ps1”を実行するための”.bat”を用意する　（現場ではこちらがおすすめ）
  • Windowsの設定を変更する　（割愛）

コマンドについて
コマンドの別名（エイリアス）が用意されており、”alias”または”get-alias”と入力すると確認できる
例 : cd(エイリアス) -> Set-Location（PowerShellコマンド）

■テキストエディタ（サクラエディタで説明）
おすすめの設定

  • タブバーを表示（設定→タブバーを表示をクリック）
    いちいちウィンドウ切り替えをしなくて済むので便利
  • カスタムメニュー（右クリック時のメニュー）によく使う機能を追加（設定→共通設定→カスタムメニュー）
    例えばSJISで開き直す / UTF-8で開き直すを追加する
    ログが同じテキストファイルに出力される場合、F5では最新の状態に更新されないので便利

おすすめの機能

  • Grep
    指定したディレクトリ内の複数のテキストファイルの中から検索や置換ができる
    「前にも似たようなことを解決してテキストファイルに書いたな！どこに書いたっけ？」というときに便利
  • キーマクロ
    記憶した文字の一括変換のルーティーンを実行できる
    文字をいじるだけならExcelマクロより軽くて簡単なので便利

その他
ヘルプからコマンド一覧やキー割り当て一覧が見られる
